## About This Project

You are looking at it! The Markuxt Portal is the project's own website, serving as both documentation landing page and a live demonstration of the framework's capabilities.

### What It Demonstrates

- All five content types: members, publications, projects, positions, news
- Bilingual i18n (English and Chinese)
- Homepage with carousel, feature highlights, and content sections
- Dark/light mode toggle
- Responsive navigation with mobile menu

### Technology

- Built with Markuxt (extending itself as a Nuxt layer)
- Styled with the Seaside theme preset
- Content authored entirely in Markdown
