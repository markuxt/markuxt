## Ways to Contribute

Markuxt welcomes contributions from developers, designers, and technical writers. Here are some areas where you can help:

### Code Contributions

- Fix bugs and implement new features
- Improve component accessibility and performance
- Add new content type support

### Theme Development

- Design new theme presets using CSS custom properties
- Improve existing theme palettes
- Create theme preview tools

### Documentation

- Write tutorials and getting-started guides
- Improve API documentation
- Translate documentation to additional languages

### Getting Started

1. Fork the [Markuxt repository](https://github.com/markuxt/markuxt)
2. Clone your fork and install dependencies with `pnpm install`
3. Start the dev server with `pnpm dev`
4. Make your changes and submit a pull request
