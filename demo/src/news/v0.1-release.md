## Markuxt v0.1 is Here

The first public release of Markuxt is now available. This release includes all core features needed to build a complete academic portal.

### What's Included

- Complete theme layer with 4 preset palettes (Seaside, Forest, Sunset, Slate)
- Markdown content types: members, publications, projects, positions, news
- Bilingual i18n support (English and Chinese)
- Dark/light mode toggle with OS preference detection
- LaTeX math rendering (KaTeX)
- Mermaid diagram support
- Code syntax highlighting via Shiki
- Responsive design with mobile navigation
- Image carousel on homepage
- Custom 404 page

### What's Next

We are working on additional features including more theme presets, search functionality, and improved documentation. Stay tuned for updates!
